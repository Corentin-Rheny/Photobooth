# Installation iPad

## Lien de l'application

https://corentin-rheny.github.io/Photobooth/

## Installation sur iPadOS 18

1. Ouvrir le lien dans Safari.
2. Appuyer sur Partager.
3. Choisir Ajouter a l'ecran d'accueil.
4. Activer Ouvrir comme app web si l'option est proposee.
5. Lancer PhotoBooth depuis l'ecran d'accueil.
6. Autoriser la camera.
7. Ouvrir Reglages dans l'app et choisir la camera USB / capture HDMI.

## Cablage camera externe

```text
Lumix S1 -> HDMI -> carte capture HDMI USB-C UVC -> hub USB-C Power Delivery -> iPad
```

Reglages conseilles du Lumix S1 : HDMI 1080p, mode manuel, 1/100 ou 1/125, f/5.6, ISO 800 a 1600, balance des blancs fixe.

## Canon SELPHY CP1500

1. Connecter l'iPad et la SELPHY CP1500 au meme Wi-Fi.
2. Dans PhotoBooth, faire une session test.
3. Appuyer sur Imprimer AirPrint.
4. Choisir Canon SELPHY CP1500.
5. Choisir papier 10x15 / 4x6 si propose.

## Hebergement NAS

Les fichiers du repository peuvent etre copies dans le dossier partage `Webapp photobooth` de ton NAS. Pour la camera sur iPad, utilise de preference une URL HTTPS servie par le NAS.
