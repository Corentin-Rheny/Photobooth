PhotoBooth iPad

Web app gratuite pour iPadOS 18 : caméra externe USB/HDMI si disponible, compte à rebours, montage 10×15 cm et impression AirPrint vers Canon SELPHY CP1500.

À utiliser depuis une adresse HTTPS. Une page ouverte directement depuis Fichiers peut bloquer la caméra.

Réglages

Si la carte capture HDMI est branchée, elle doit apparaître comme « USB Camera », « Capture » ou nom similaire.

Tirage prêt

Dans AirPrint : choisir « Canon SELPHY CP1500 », 1 copie, papier 10×15 / 4×6 si proposé.

Installation rapide sur iPad

  1. Héberge ce dossier sur une adresse HTTPS gratuite, par exemple GitHub Pages, Netlify Drop, Cloudflare Pages ou ton NAS en HTTPS.
  2. Sur l’iPad : ouvre l’adresse dans Safari.
  3. Touche Partager → Ajouter à l’écran d’accueil.
  4. Active « Ouvrir comme app web » si l’option apparaît, puis touche Ajouter.
  5. Branche la carte capture HDMI USB-C, puis appuie sur « Démarrer la caméra ».
  6. Va dans Réglages → Caméra → choisis la caméra USB/capture.
  7. Connecte l’iPad et la SELPHY CP1500 au même Wi‑Fi, puis teste l’impression.

Câblage option B

Lumix S1 → HDMI → carte capture HDMI USB-C → hub USB-C PD → iPad
iPad → Wi‑Fi → Canon SELPHY CP1500

L’impression silencieuse complète n’est pas possible dans une web app iPad : AirPrint demande une validation.
